Given Name Surname Job Title Department Cost

Answers

  1. Question 1:
    SELECT emp.first_name AS "Given Name", emp.last_name AS "Surname", emp.salary AS "Cost" FROM employees emp;
  2. Question 2:
    SELECT * FROM employees emp JOIN departments dep ON emp.department_id = dep.department_id;
  3. Question 3:
    SELECT * FROM employees emp WHERE emp.salary > 5000;
  4. Question 4:
    SELECT * FROM employees emp WHERE emp.salary > 5000 ORDER BY emp.salary ASC;
  5. Question 5:
    SELECT emp.first_name AS "Given Name", emp.last_name AS "Surname", job.job_title AS "Job Title", dep.department_name AS "Department", emp.salary AS "Cost" FROM employees emp JOIN departments dep ON emp.department_id = dep.department_id JOIN jobs job ON emp.job_id = job.job_id WHERE emp.salary > 5000 ORDER BY emp.salary;
